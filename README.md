Nuestro Diario Digital: Aplicación personalizada para compartir momentos y mensajes. Desarrollada en React con persistencia de datos en la nube mediante Firebase para una experiencia interactiva en tiempo real.
# Anniversary App - John & Cristian

Una aplicación web progresiva (PWA) diseñada para capturar momentos especiales. 

### Tecnologías
* **Frontend:** React.js + CSS Modules.
* **Backend:** Firebase (Firestore para mensajes, Cloud Storage para fotos).
* **Deployment:** Vercel / GitHub Pages.
* **Mobile:** Configuración PWA para instalación nativa en iOS/Android.

### Funcionalidades
- [x] Sincronización de fotos en tiempo real.
- [x] Panel de mensajes interactivo.
- [x] Instalación en pantalla de inicio (Add to Home Screen).
